Q6. Explore: 

Can the same method name be used for different methods without any conflict?
(Answer)
Yes the same method name can be used for different methods without conflict through a concept known as method overloading. It allows multiple methods to have the same name but can differ in the number or type of their parameters. This feature is available in Java.
